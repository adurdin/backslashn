Find text in MySQL database

I had an email address in an error log. It was coming from this MySQL database, somewhere, but I had no clue where.

But it was easy to find out where:

mysqldump --opt --skip-extended-insert --complete-insert -uusername -p database_name | grep 'email@example.com'

This produces a single INSERT statement, showing me exactly where the email address is to be found: table, column, and primary key.
